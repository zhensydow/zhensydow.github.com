Herramientas Usadas

Ubuntu / Mint
Las distribuciones de Linux que hemos usado para generar todo lo necesario para la boda.
LaTeX
Un sistema de preparación de documentos. Permite escribir documentos describiendo el contenido y la forma y LaTeX realiza la composición de página y genera el documento final.
Inkscape
Programa de diseño vectorial, permite preparar facilmente posters, combinando imagenes, figuras geometricas y texto de manera visual.
Gimp
Programa de edicion y retoque fotográfico.

Música

Para la ceremonia habiamos seleccionado una lista de canciones que iban desde música clásica e instrumental para el recibimiento, rock suave, años 60~70 para la comida y algo más movida a medida que avanzaba la fiesta hasta llegar al reggeton si se hubiese alargado mucho el día. Algunos respiraron tranquilos de no llegar a este punto. La lista completa con el orden se puede ver en Música de la Boda.
